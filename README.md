# 🍅 뽀모도로 타이머 (Pomodoro Timer)

수능·전공 공부에 딱 좋은 미니멀한 뽀모도로 타이머. 눈이 피로하지 않도록 차분한 종이 질감 배경과 부드러운 파스텔 톤으로 디자인했습니다.

## 주요 기능

- **자동 모드 전환** — 집중이 끝나면 휴식으로, 휴식이 끝나면 다시 집중으로 자동 전환
- **시간 직접 조절** — 그날 컨디션·과목에 맞게 집중/휴식 시간을 1~120분 범위로 설정 (설정값은 브라우저에 저장)
- **세션 카운터** — 오늘 완료한 집중 사이클을 기록 (하루 단위 초기화)
- **원형 진행 링** — 남은 시간을 시각적으로 표시
- **상태별 컬러 UI** — 집중은 로즈 톤, 휴식은 블루 톤
- **종료 알림음** — 세션이 끝나면 짧은 알림음 재생

## 기술 스택

React 18 · Vite 6 · Tailwind CSS 4 · lucide-react

## 로컬 실행

```bash
npm install
npm run dev
```

## 빌드

